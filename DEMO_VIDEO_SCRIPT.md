## Recording Tips

1. **Resolution:** 1920x1080 minimum (recorded on desktop, full screen)
2. **Speed:** Normal speed (no sped-up segments)
3. **Audio:** Clear voiceover, no background noise
4. **Pacing:** Deliberate, not rushed — let people see the animations
5. **URL:** Include the live Vercel URL in the video description (e.g., "https://varavault-123.vercel.app")
6. **End card:** Final 5 seconds should show the live URL + GitHub link
